Tunelito live annotation test

A simple page for checking shared comments.

This version intentionally avoids any custom review JavaScript. Select text anywhere on the page, use the Tunelito comment bubble, and the comment should appear back on the host machine.

What to Try First

Select this sentence and leave a short comment from your other device. If the live sync is working, the host terminal should log the comment and a markdown file should appear beside this HTML file.

The page keeps the same public URL because the local server is still serving the same filename. Only the file contents changed.

Review Notes

Tunelito is meant for quick calls where one person owns the file and everyone else can leave feedback. The author edits locally, viewers refresh automatically, and the comments remain as a plain markdown artifact.

This test page has ordinary headings, paragraphs, and cards so we can verify text selection, anchored comments, sidebar updates, and markdown persistence without another annotation system interfering.
