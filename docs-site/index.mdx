---
title: "Tunelito"
description: "Turn local HTML into a temporary live review room with comments."
---

Tunelito turns any local HTML file or folder of HTML files into a temporary live review room.

Run one command, share the printed link on a call, and reviewers can select text, leave page notes, or leave site-wide notes live. You keep editing the HTML in your editor. Connected browsers reload when files change. Comments are saved as readable Markdown by default, or kept ephemeral with `--live`.

## Why it exists

Sometimes the fastest way to review a draft, prototype, spec, or one-off HTML page is not to deploy an app. It is to point at the file, get a link, and talk through it with someone.

Tunelito gives you that loop:

- local HTML stays the source of truth
- reviewers can comment but cannot edit the files
- comments sync live, with WebRTC peer-to-peer support in `--live`
- folder reviews share one Markdown inbox with page and site comment scopes
- an opt-in local agent worker can call Codex, Claude Code, or another CLI to handle comments
- public sharing uses a temporary Cloudflare Tunnel
- shared links require a generated review key by default

## Start

```bash
npx --yes tunelito ./page.html
```

For a folder:

```bash
npx --yes tunelito ./site
```

Follow the [folder review tutorial](/tutorial) to collect comments across multiple HTML pages and have a local coding agent act on the Markdown inbox.

For mobile review sites people save to an iPhone Home Screen, use the [iPhone Home Screen guide](/iphone-home-screen) to keep internal navigation inside the saved web app.

For unattended comment follow-up:

```bash
npx --yes tunelito ./site --agent codex
```

## What you share

Tunelito prints a local URL and, when tunneling is available, a public URL:

```text
Tunelito is running
Local:   http://127.0.0.1:4317/?tunelito_key=...
Comments: /path/to/page.comments.md
Access:  review key required by the printed URLs
Public:  https://example.trycloudflare.com/?tunelito_key=...
```

Share the `Public:` URL with the person on your call.

For ephemeral in-meeting collaboration:

```bash
tunelito ./page.html --live
```
