---

The repo is designed for agent-driven maintenance.

## Core checks

```bash
npm run check
npm run agent:check
npm test
npm run ci
```

## Package smoke

```bash
npm run pack:check
```

This packs the current tree, installs the tarball into a clean global prefix,
and verifies both the installed `tunelito` binary and an `npx --package`
execution path.

## npm install smoke

After publishing:

```bash
npx --yes tunelito --version
```
