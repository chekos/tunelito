Tunelito includes example HTML pages in the GitHub repository.

## Simple review page

Use this for normal smoke tests:

```bash
tunelito ./examples/simple-review.html --no-tunnel --open
```

The page contains styled DOM text and no custom review JavaScript.

## Legacy prototype page

`examples/legacy-data-architecture-review.html` preserves the original self-contained review prototype that inspired Tunelito.

It is useful as a stress or regression fixture because it already has its own review UI and a lot of embedded JavaScript.

For normal beta demos, prefer `simple-review.html`.
